9. Таланты
Для проверки талантов используется следующий формат записи:
talent.имя_или_ID.параметр
talent — проверка таланта
имя_или_ID — имя таланта в формате программы или числовой ID способности
параметр — что именно нужно проверить: выбран ли талант или какой у него ранг
Если параметр не указан, то проверяется просто факт наличия таланта. Это то же самое, что и .enabled.
ВЫРАЖЕНИЕ ЧТО ВОЗВРАЩАЕТ ПРИМЕР
talent.ИМЯ_или_ID Талант выбран if=talent.avatar
talent.ИМЯ_или_ID.enabled То же самое, что и talent.ИМЯ_или_ID if=talent.107574.enabled
talent.ИМЯ_или_ID.rank Ранг таланта if=talent.avatar.rank>=1
Проверка по ID может быть полезна, если вы хотите использовать точное числовое значение способности.
# Использовать Avatar только если этот талант выбран
actions+=avatar,if=talent.avatar
# Использовать Shockwave только если взят соответствующий талант
actions+=shockwave,if=talent.shockwave.enabled
# Проверка таланта по ID
actions+=storm_bolt,if=talent.107570
