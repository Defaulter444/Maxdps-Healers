17. Сбитие кастов (Kick / Interrupt)
Открывается кнопкой ⚡  KICK в нижней панели (у ДПС-профилей). Программа будет автоматически сбивать касты врагов
согласно вашим настройкам.
Режимы сбивания
РЕЖИМ ПОВЕДЕНИЕ
Не сбивать Сбивание отключено полностью
Сбивать все касты Любой каст врага будет прерван
Только Whitelist Сбивать только способности из белого списка (по ID)
Все, кроме BlacklistСбивать всё, кроме способностей из чёрного списка (по ID)
Whitelist и Blacklist
Оба списка содержат ID способностей WoW — по одному на строку. ID можно найти на Wowhead или через аддон.

Пороги прерывания
НАСТРОЙКА ОПИСАНИЕ
Cast interrupt %На каком проценте прогресса каста прерывать. 0% = сразу, 80% = ближе к концу. По умолчанию: 20%
Channel interrupt %То же самое для чаннелируемых способностей. По умолчанию: 20%
Совет:
Значение ~20% означает «сбивать когда каст прошёл 20%». Это имитирует реальное поведение игрока — чуть подождать чтобы
убедиться что каст не отменится сам, и затем прервать.
После изменения настроек нажмите APPLY — настройки сохраняются в текущем профиле ротации.
