Структура строки
Базовый формат
В общем виде использование способности с условием выглядит следующим образом:
actions+=название_способности,if=условие
actions — обязательное ключевое слово
название_способности — имя способности
if=условие — условие применения (необязательно)
Например, если нужно использовать Mortal Strike просто по КД:
actions+=mortal_strike
Если же вы хотите использовать Mortal Strike только тогда, когда у вас больше 50 ярости:
actions+=mortal_strike,if=rage>50
Вот из таких базовых строк и строится ротация — последовательность действий с условиями, идущих друг за другом.
Программа берет строки по порядку и проверяет условия, которые в н них указаны. Если они выполнились — способность
применяется. Если нет — программа переходит к следующей строке.
Например, простая ротация может выглядеть так:
# Использование Mortal Strike просто по готовности
actions+=mortal_strike
# Используем Rend, если на цели нет дебафа
actions+=rend,if=!target.debuff.rend.up
# Используем, если до отката Mortal Strike больше 3 секунд или у нас более 50 ярости
actions+=slam,if=cooldown.mortal_strike.remains>3|rage>50
В этом примере сначала проверяется Mortal Strike. Если способность готова, она используется без дополнительных
условий. Если же Mortal Strike недоступен (например, он на кулдауне или для него не хватает ярости), программа
переходит к следующей строке и проверяет условия для Rend. Если на цели нет дебафа, то Rend применяется. Если дебаф
уже есть, программа переходит к проверке Slam, который будет использоваться, если до готовности Mortal Strike
больше 3 секунд или у вас больше 50 ярости.
Комментарии
Как вы могли заметить, некоторые строки начинаются с символа #. Это знак комментария. Все строки, начинающиеся с
этого символа, игнорируются программой и используются только для пояснений.
# Это комментарий — его программа не читает
actions+=mortal_strike,if=rage>=30
# Следующая строка — финишер на низком здоровье
actions+=execute,if=target.health.pct<20
Названия способностей
Названия способностей можно взять прямо из игры (если используется англоязычный клиент) или посмотреть на WoWhead.
Поддерживаются только английские названия, приведённые к формату: все буквы строчные, пробелы заменены на символ
подчеркивания _, а специальные символы удалены.
