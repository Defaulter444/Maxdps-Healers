19. Выбор членов группы и фокус-цель Хилер
Открывается кнопкой ⌨  HOTKEYS. Для хилерских профилей дополнительно отображаются слоты для группы.
Программа может автоматически переключать цель на нужного члена группы чтобы применить к нему хил. Для этого
каждому члену нужно назначить клавишу — программа «нажмёт» её в нужный момент точно так же как это делаете вы.
Как назначить клавишу
1. Нажмите на нужный слот (P1–P5, R1–R30, или Focus)
2. Слот станет оранжевым с «● key...»
3. Нажмите клавишу (с Ctrl / Shift / Alt или без)
4. Escape — отменить. Delete — очистить клавишу
Доступные слоты
СЛОТ КОЛИЧЕСТВО ОПИСАНИЕ
Focus 1 Клавиша для фокус-цели. Программа нажмёт её чтобы выбрать вашего фокуса как цель хила
P1 – P55 Клавиши для членов группы (Party). P1 = первый член группы, и т.д.
R1 – R3030 Клавиши для членов рейда (Raid). R1 = рейд-член 1, и т.д.
Совет:
Используйте встроенные WoW макросы для переключения целей. Создайте в игре макросы вида /target party1, /target
raid1 и привяжите их к клавишам. Затем эти же клавиши укажите в слотах P1–P5 и R1–R30.
Важно:
Если для члена группы не назначена клавиша, программа не сможет переключиться на него как на цель хила. Этот член будет
пропускаться при автовыборе цели.
